# Contact Cards Application

A React application that displays contact information using reusable card components. The project demonstrates the use of props and component reusability by rendering multiple contact cards from a single component structure.

## Features

- Reusable ContactCard component
- Multiple contact profiles
- Responsive card layout
- Clean and modern design
- Dynamic rendering using props

## Technologies Used

- React
- JavaScript
- CSS
- Vite

## Concepts Demonstrated

- Functional Components
- Props
- Component Reusability
- Array Mapping
- Responsive Design

## Live Demo

https://contact-card-sooty.vercel.app/

## Learning Outcomes

This project helped me understand how data can be passed between components using props and how reusable components improve code organization.
